# Landing Page Optimization Checklist

This document contains the comprehensive rubric for scoring landing pages.

## Scoring System
- **High Priority**: 3 points
- **Normal Priority**: 2 points
- **Optional**: 1 point

**Total Possible Score**: 198 points
